
# Trenda

This is a Next.js code bundle for Trenda. The original project is available at https://www.figma.com/design/8BhY1AfA4FoJf1mC233H4a/Trenda.

## Running the code

1. Run `npm install` to install the dependencies.
2. Run `npm run dev` to start the Next.js development server on http://localhost:3000.
